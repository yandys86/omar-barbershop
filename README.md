# Omar Barbershop — Landing

Barbería premium de estilo cubano en Houston (Bellaire Blvd). Web one-page con animaciones al scroll, calendario de reservas y reseñas reales de Google.

🌐 **Live:** https://yandys86.github.io/omar-barbershop/

## Stack

Un único `index.html` autocontenido (HTML + CSS + JS inline). Cero dependencias externas salvo Google Fonts. Funciona abriendo el archivo directamente o sirviendo cualquier estático.
